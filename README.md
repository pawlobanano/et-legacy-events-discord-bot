## ET: Legacy Events Discord bot
A helper tool for tournament management.

### Bot keywords
```
!cup help | h
!cup teams | t
(todo) !cup team <a,b,c,...>
!cupbot status | s
```

### Run bot
```sh
make run
make binary
```

### API Credentials
Required files:
- key.json
- .env
